# CLI Todo Manager

Advanced command-line todo application written in Python.

## Features
- Priorities (low / medium / high)
- Deadlines with overdue highlighting
- Persistent JSON storage
- Rich terminal UI
- Installable via pip
- Tested with pytest

## Installation
```bash
pip install -e .

## USAGE

- todo add "NEW task" --priority high --deadline 2026-01-15
- todo list
- todo done 1
- todo delete 1
